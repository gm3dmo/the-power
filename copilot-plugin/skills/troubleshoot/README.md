# troubleshoot

Diagnose SSL errors, token issues, HTTP failures, and config problems in the-power.

## When to use

When a script fails, returns an unexpected HTTP status, or the environment is not working as expected.

## Quick start

```bash
cd the-power
./get-authenticated-user.sh       # test token validity
./get-an-organization.sh          # test org access
```
