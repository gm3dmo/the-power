# create-test-org

Spin up a fully populated test organisation with users, teams, and repos on a GHES instance.

## When to use

When you need a complete test environment with an org, team, users, repository, PRs, branch protection, and releases.

## Quick start

```bash
cd the-power
python3 configure.py           # if not already configured
./build-all.sh
```
