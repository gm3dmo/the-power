# create-test-repo

Create a test repository with the right testcase for your scenario — PRs, workflows, secret scanning, rulesets, and more.

## When to use

When you need a test repo that matches a specific customer scenario or feature area.

## Quick start

```bash
cd the-power
./build-testcase                          # basic repo with PR
./build-testcase-workflow-simple          # GitHub Actions
./build-testcase-secret-scanning          # secret scanning
```
