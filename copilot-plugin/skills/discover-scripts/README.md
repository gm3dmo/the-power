# discover-scripts

Find the right script from the-power's ~980 shell scripts by category, keyword, or goal.

## When to use

When you or an agent needs to find a script for a specific GitHub API operation — create, list, get, delete, update, or orchestrate.

## Quick start

```bash
cd the-power
ls *.sh | grep -i "webhook"          # find webhook scripts
ls *.sh | grep -i "branch-protect"   # find branch protection scripts
ls build-testcase*                    # list all testcase orchestrations
```
