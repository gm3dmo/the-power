# run-script

Execute any the-power script with guided parameter input and output interpretation.

## When to use

When you need to run a specific script and understand its output, error codes, or debug failures.

## Quick start

```bash
cd the-power
./create-repo.sh              # create a repo
./list-organization-members.sh # list org members
./build-testcase              # full testcase
```
